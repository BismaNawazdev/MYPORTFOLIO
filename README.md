# Personal Portfolio Website 💻✨

Welcome to my personal portfolio repository! This is a modern, responsive, and dark-themed portfolio website built from scratch. It serves as my digital resume to showcase my technical skills, programming journey, and the web development projects I have built as a Computer Science student.

## Tech Stack Used 🛠️
* **HTML5:** For structured, semantic, and SEO-friendly web architecture.
* **CSS3:** For custom modern typography, neon styling, grid/flexbox layouts, and a clean dark-mode user interface.
* **JavaScript (ES6):** For smooth scrolling, dynamic animations, and interactive elements.

## Skills & Technologies Featured 🚀
* **Front-End Development:** HTML5, CSS3, JavaScript (ES6), React Basics, Tailwind CSS Basics
* **Programming & Logic:** C, C++, C#, Java, Problem Solving
* **Databases:** SQL
